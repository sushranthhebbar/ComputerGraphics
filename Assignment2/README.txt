How to Run?

Open in Visual Studio Code. Install Live Server plugin. Right click on source/code/index.html and click on Run Live Server.

Controls:-
The M key is used to cycle across different modes. There are 4 modes. Mode 0 is instance
transformation mode. The arrow keys are used to move the selected object in the x-y plane.
The I and O keys move the primitive along the z axis. The numpad + and numpad - keys scale
the selected primitive. The numpad 8 and numpad 2 rotate the primitive by a positive and
negative angle respectively. The A key is used to cycle across the 3 different axes. Mode 1 is
Camera transformation mode. Click the left mouse button and drag left to right or the reverse
to move the camera along the selected axis. Mode 2 is object mode. Clicking on the primitive
changes the color of the whole primitive in a cyclic manner. Mode 3 is face mode. Clicking on
the face of the primitive changes the color of the face in a cyclic fashion. Clicking anywhere
else unselects the clicked face or object.
